Scratch 3D Extension Kit

This GitHub Pages site hosts the public Gandi test bundle for the shared 3D extension scaffold.

./gandi-normal-remote.js
Open JS bundle Open Gandi

For Gandi custom extension testing: copy the full URL ending in gandi-normal-remote.js into the custom-extension input, or append it to the ?gext= parameter.

For TurboWarp: keep using the local dev server and http://localhost:8000/turbowarp-unsandboxed.js. TurboWarp unsandboxed public URLs are intentionally restricted.
